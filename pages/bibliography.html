Annotated Bibliography

Web Development

These resources relate to the coding and development of the website. I used them to learn popular web development technologies such as HTML, CSS, JavaScript, React, and Canvas API. I used The Odin Project for complete tutorials to give me the foundations of each tool. I used MDN to find specific information about each tool as needed.

  • "Canvas API." MDN. https://developer.mozilla.org/en-US/docs/Web/API/Canvas_API.
  • "JavaScript frameworks and libraries." MDN. https://developer.mozilla.org/en-US/docs/Learn_web_development/Core/Frameworks_libraries
  • MDN. https://developer.mozilla.org/en-US/.
  • The Odin Project. https://www.theodinproject.com/.

Website Hosting

These resources detail how to initialize and manage a web server. I loaned out physical server resources and purchased a domain name, so I linked and configured the two to form a functioning web server. I downloaded nginx onto the server as the web-server hosting software. DigitalOcean is the company that owns the servers, so I used their documentation to connect the domain name to their server.

  • DigitalOcean Docs. https://docs.digitalocean.com/.
  • nginx. https://nginx.org/.

User Interface & User Experience

These resources relate to the user interface and user experience design of the website. I used them to learn fundamental web design principles. The books "Don't Make Me Think" and "Laws of UX" taught me accessible and user-friendly design, while "Refactoring UI" taught me aesthetic design.

  • Krug, Steve. Don’t Make Me Think Revisited: A Common Sense Approach to Web and Mobile Usability. New Riders, 2014.
  • Laws of UX. https://lawsofux.com/.
  • Wathan, Adam, and Steve Schoger. Refactoring UI.
